PICO-200S Manual Table Feed Precision Wafering Saw

The PICO-200S Manual Precision Saw is ideal for metallographic cutting of small samples. With adjustable speeds (500–3000 rpm), support for 6–8" blades, and manual control, it ensures precision and versatility. Features include immersion cooling, speed display, and dual vises—all in a compact, durable design.

Features

  • Compact Design: Space-saving dimensions ideal for laboratory use
  • High Cutting Capacity: Supports up to 1-inch (30 mm) cutting capacity
  • Efficient Specimen Feed: Features a table feed for smooth operation
  • Powerful Motor: Equipped with a 400 W motor for robust performance
  • Variable Speed Control: Continuously adjustable speed (500-3000 rpm) with electronic display
  • Integrated Coolant System: 3-liter coolant tank for effective heat management
